🎮 SSE Single Session Test

Herramienta de prueba para el sistema de sesión única. Abre esta página en múltiples pestañas/ventanas para simular diferentes dispositivos.

Configuración de Conexión

📝 Instrucciones de Prueba:

  • Paso 1: Abre esta página en 2 o más pestañas del navegador
  • Paso 2: En cada pestaña, usa el MISMO "Manager ID" (simula mismo usuario)
  • Paso 3: Conecta desde la primera pestaña - verás "Conectado"
  • Paso 4: Conecta desde la segunda pestaña - verás cómo la primera recibe "session_terminated"
  • Paso 5: Prueba "Desconexión Forzada" para cerrar todas las sesiones del usuario
